Verbal Autopsy Coding — How It Works
  1. The stats cards separate Random Allocation work from Pick And Choose work so you can see exactly which intake mode is available to you.
  2. Click "Start Random Allocation Coding" to let the system assign you a case from projects that use random allocation. Once assigned, complete its coding before picking another. If a VA form is already in progress, you'll see "Continue Pending VA Coding" instead. Normal coding allocations are revoked after 1 hour if not completed. Demo/training coding allocations are revoked after 15 minutes.
  3. Use the "Pick And Choose Coding" table only for projects that require manual case selection.
  4. Use the "View" button to open any completed VA form in read-only mode.
  5. Use the "Recode" button to update a coded VA form. A form can only be recoded once, and only within 24 hours of its initial coding.
  6. For assistance, contact the Central Admin Team at AIIMS, New Delhi.
